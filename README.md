# FPGA-Red-light-Green-light-game-

FPGA implementation of Red Light, Green Light on the Nexys A7 — 4-player game with FSM-based game logic, iterative timing, elimination detection, and 7-segment display output, written in VHDL. 

.do and test bench files are only needed for simulation on modelsim. When simulating on Modelsim, all integer signals related to clock cycles should be reducde by a factor of 10^5 or else the simulation will take too long.
